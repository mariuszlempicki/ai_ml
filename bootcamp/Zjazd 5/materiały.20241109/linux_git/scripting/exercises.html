    Zadanie 1: instalator

    1. Stwórz skrypt display.sh wypisujący zawartość wszystkich plików z podkatalogu data znajdującym się katalogu bieżącym
    2. Napisz skrypt instalacyjny setup.sh
      • skrypt powinien obsługiwać dwie opcje -i oraz -u
    3. Opcja -i powinna pobierać argument będący ścieżką do zainstalowania skryptu display.sh
      • Skrypt powinien skopiować plik display.sh oraz utworzyć katalog data w lokalizacji wskazanej jako argument
      • W katalogu data powinien stworzyć dwa pliki:
        • pierwszy powinien zawierać nazwy wszystkich plików zaczynających się literą "b" z katalogu /usr/bin
        • drugi powinien zawierać nazwy wszyskich plików rozpoczynających się literą "p" z katalogu /etc
    4. Opcja -u powinna pobierać argmument będący ścieżką do zainstalowanej aplikacji
      • Skrypt powinien usuwać skrypt display.sh oraz katalog data
      • Jeśli po usunięciu powyższych plików ścieżka jest pusta powinien również usunąć katalog nadrzędny

    Zadanie 2: Budzik

    Stwórz skrypt pobierający jeden argument będący ilością sekund do uruchomienia budzika. Budzik powinien działać w następujący sposób:

    • Po upływie zadeklarowanego czasu powinien wyświetlić alert przy pomocy notify-send oraz odtworzyć dźwięk przy pomocy aplay 1)
    • Jeśli przed upływem czasu otrzymał sygnał SIGINT lub SIGTERM, powinien wyświetlić komunikat przy pomocy notify-send

    1) Można użyć któregoś pliku .wav z dźwiękami systemowymi.

    Opcjonalne modyfikacje

    • Zmień skrypt tak, aby zamiast ilości sekund do uruchomienia budzika przyjmował godzinę o której należy go uruchomić
