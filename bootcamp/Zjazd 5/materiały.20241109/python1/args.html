    Przekazywanie argumentów do funkcji

    Do funkcji w Pythonie możemy przekazywać argumenty nazwane (ang. keyword arguments) lub argumenty nienazwane.

    def print_n(text, n):
    for i in range(n):
        print(text)

print_n("my text", 3) #  (1)!
print_n("my text", n=3) #  (2)!
print_n(text="my text", n=3) #  (3)!
    1. Argumenty pozycyjne
    2. Argument pozycyjny i argument nazwany
    3. Argumenty nazwane

    Danger

    Argumenty pozycyjne nie mogą występować po nazwanych

    print_n(text="my text", 3)

      File "/path/to/code/python1/fun_args.py", line 6
    print_n(text="my text", 3)
                         ^
    SyntaxError: positional argument follows keyword argument

    Wartości domyślne

    W definicji funkcji argumentom można nadać wartości domyślne.

    def print_n(text, n=3):
    for i in range(n):
        print(text)

print_n("my text") #  (1)!
print_n("my text", 5) #  (2)!
print_n("my text", n=5) 
print_n(text="my text", n=5)
    1. n=3
    2. n=5

    Danger

    Nie należy używać obiektów zmiennych (ang. mutable) jako wartości domyślnych dla argumentów funkcji.

    def append_list(item, lst=[]):
    lst.append(item)
    return lst

l = [1]
append_list(2, l)
print(l)
append_list(3, l)
print(l)

print('-' * 10)
m = append_list('a')
print(m)
n = append_list('b')
print(n)

    [1, 2]
[1, 2, 3]
----------
['a']
['a', 'b']

    def append_list(item, lst=None):
    if lst is None:
        lst = []
    lst.append(item)
    return lst

l = [1]
append_list(2, l)
print(l)
append_list(3, l)
print(l)

print('-' * 10)
m = append_list('a')
print(m)
n = append_list('b')
print(n)

    [1, 2]
[1, 2, 3]
----------
['a']
['b']

    Zmienna ilość argumentów

    Python pozwala na definiowanie funkcji ze zmienną liczbą argumentów. W tym celu zostały wprowadzone dwa operatory * dla argumentów nienazwanych i ** dla argumentów nazwanych.

    Operator * (*args)

    def my_func(*args):
    print(f'{args=}')
    print(f'{type(args)=}')

my_func(1, 2, 3)

    def my_func(*args):
    for arg in args:
        print(arg)

my_func(1, 2, 3)

    def my_func(arg1, *args):
    print(f'{arg1=}')
    print(f'{args=}')

my_func(1, 2, 3)

    Operator ** (**kwargs)

    def my_func(**kwargs):
    print(f'{kwargs=}')
    print(f'{type(kwargs)=}')

my_func(a=1, b=2, c=3)

    def my_func(**kwargs):
    for name, value in kwargs.items():
        print(f'{name=}, {value=}')

my_func(a=1, b=2, c=3)

    def my_func(arg1, **kwargs):
    print(f'{arg1=}')
    print(f'{kwargs=}')

my_func(1, b=2, c=3)
my_func(arg1=1, b=2, c=3)

    Konwersja krotek i słowników na argumenty funkcji

    values = (1, 2, 3)
kvalues = {'a': 10, 'b': 11, 'c':12}


def my_func(*args, **kwargs):
    for arg in args:
        print(arg)
    for name, value in kwargs.items():
        print(f'{name=}, {value=}')

my_func(*values, **kvalues) #  (1)!
    1. my_func(1, 2, 3, a=10, b=11, c=12)
